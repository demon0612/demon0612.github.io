# 一、标题

### 语法：

每一即标题前面加#号。#号越多，级数越高。
注意：#号和标题内容之间要有空格。

### 实例：

```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

###  效果：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

# 二、字体样式

### 语法：

共四种：加粗、倾斜、加粗倾斜、删除线。注意：字体两侧不能有空格！

### 实例：

```
**加粗**
*倾斜*
***加粗倾斜***
~~删除线~~
```

###  效果：

**加粗**

*倾斜*

***加粗倾斜***

~~删除线~~
